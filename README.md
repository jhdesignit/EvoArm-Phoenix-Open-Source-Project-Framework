WHAT IS THE EVOARM PHOENIX?

The EvoArm Phoenix is a high-performance, medical-grade upper limb prosthetic platform built to deliver advanced bionic functionality to recipients worldwide completely free of charge.

The system runs on an ESP32 microcontroller hosting a localized, asynchronous web server that broadcasts a dedicated portrait Web UI for real-time calibration. Biological control is established via non-invasive surface electromyographic (EMG) sensors that map muscle signal inputs to a fluid 10-grip target matrix.

To ensure absolute field-reliability, the internal power architecture completely isolates high-current motor pathways from sensitive microcontroller logic via dedicated adjustable buck converters. All power, sensor, and control channels route through an integrated wire harness into a centralized 15-pin block connector, safely enclosed within an internal electronics housing shell to prevent friction and wire insulation fatigue.

Our global tracking updates utilize strict anonymous data sets paired with conceptual visualization to fully protect the real-world identity of our young users while celebrating their technical development milestones.

MANUFACTURE INTEGRITY NOTICE

To protect internal electronic traces from short-circuit paths and abrasive print friction, the workshop enforces a strict Full-Integrity Finishing Rule across all three download packages. No internal electronics wiring, 15-pin blocks, or mechanical actuators may be mounted inside the forearm shell until the 3D-printed internal compartments have been fully sanded, primed, painted, and sealed with a protective lacquer coat.
