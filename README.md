# EvoArm-Phoenix-Open-Source-Project-Framework
Open-source repository for the EvoArm Phoenix by JH Designs. Providing free, medical-grade bionic limbs via C++ firmware, ESP32 Web UI, and SolidWorks CAD files. Our 7-day manufacturing protocol empowers families and classrooms worldwide with accessible EMG-controlled robotics. Download our beginner, intermediate, or advanced packs to get started.
